发表于 2018-01-29

python正则

常用re.findall(r'pattern',<string>),得到一个list

re 模块提供了不少有用的函数，用以匹配字符串，比如：

  • compile 函数
  • match 函数
  • search 函数
  • findall 函数
  • finditer 函数
  • split 函数
  • sub 函数
  • subn 函数

re 模块的一般使用步骤如下：

  • 使用 compile 函数将正则表达式的字符串形式编译为一个 Pattern 对象
  • 通过 Pattern 对象提供的一系列方法对文本进行匹配查找，获得匹配结果（一个 Match 对象）
  • 最后使用 Match 对象提供的属性和方法获得信息，根据需要进行其他的操作

核算

计息规则

  • 根据实际发生天数计算每日应收利息，当日计提利息= 贷款当期截止到当前的应收利息 — 当期已经记账的应收利息
  • 贷款当期截止到当前的总计提金额 = （上次还款日到当前的天数 剩余未逾期本金 合同日利率）
  • 若实际结息天数为31天，则在第30天时，当日应收利息=还款计划上的当月利息—当期已经记账的应收利息；第31天不再结转应收利息
  • 若实际结转天数小于30天，2.15—3.15，则再3.15日时，当日结转利息=还款计划上的当月利息—当期已经记账的应收利息
  • 已经核销、已经提前结清的贷款，利息计提到核销和结清当日。之后不再进行计提。（核销当日仍计提）
